# YRPGCore
RPG核心
玩法概述:開放式世界 分不同區域有不同怪物群，城鎮 每個地標有幾個副本
        
        城鎮: 1.在每個副本附近 2.到過的城鎮能解鎖傳送點 3.不同城鎮能販售主商店沒有的東西or特價 4.有npc能解支線任務
        
        副本: 1.人數1-4人 2.會掉落任務獎勵 稀有合成素材 3.要依序解鎖副本 4.等級限制
        
        職業: 
        
        
        
        武器: 
             取得:商店購買 解任務
             升級:消耗同階級武器合成
             強化:消耗強化素材 
             限定:不同職業有指定的武器類型
             
         等級:升級能獲得技能點 能限制某能力值不能太多
